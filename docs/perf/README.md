# Performance Records

This directory contains curated historical performance evidence.

Performance notes and baseline snapshots here preserve measurements from a
specific commit, machine, build type, benchmark profile, and point in time. They
are evidence for comparison, not current instructions, CI thresholds, or
permanent benchmark standards.
Archived commands may use interfaces that no longer exist; preserve them as
provenance, but do not copy them as current benchmark guidance.

Raw benchmark output belongs under `runs/`, which is gitignored. Commit only
curated summaries that have lasting comparison value.

Avoid labels such as "Latest" unless someone is actively maintaining that label.
When in doubt, describe a file by date, PR, commit, profile, and purpose.

Current benchmark and measurement guidance lives in
[`../benchmarks.md`](../benchmarks.md).
