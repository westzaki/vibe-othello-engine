## Interpretation

- Fixed-depth search with TT on is the recommended depth-10 analysis setting for
  now.
- Iterative deepening remains useful for future time-control work, but this
  snapshot shows higher cumulative cost at the same final depth.
- Future search and evaluation changes should compare against p50, p95, max, and
  node counts rather than average elapsed time alone.
